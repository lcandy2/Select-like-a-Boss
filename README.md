## Features

<table>
    <tr>
        <td><b>select</b></td>
        <td>↔</td>
        <td>like a regular text</td>
    </tr>
    <tr>
        <td><b>drag</b></td>
        <td>↕</td>
        <td>up or down</td>
    </tr>
</table>

*Notice: some links can be unselectable*
