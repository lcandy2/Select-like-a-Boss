Select like a Boss

If you want to use the extension in the incognito mode too, do not forget to enable the checkbox [Allow in incognito] on the extensions page.

Thank you for using 'Select like a Boss' extension!

Now you can select link's text just like a regular text (like in Opera'12 browser) - Select like a Boss ;)

  • when you want to select ↔ just select it like a regular text;
  • when you want to drag ↕ drag it up or down.

Enjoy! (and share the extension with other people)


Notice: some links can be unselectable.
