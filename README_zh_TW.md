## 功能

<table>
    <tr>
        <td><b>選取</b></td>
        <td>↔</td>
        <td>同普通文字一樣</td>
    </tr>
    <tr>
        <td><b>拖曳</b></td>
        <td>↕</td>
        <td>向上或向下</td>
    </tr>
</table>

*注意：有些連結可能無法選取*
