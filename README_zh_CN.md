## 功能

<table>
    <tr>
        <td><b>选择</b></td>
        <td>↔</td>
        <td>和普通文字一样</td>
    </tr>
    <tr>
        <td><b>拖拽</b></td>
        <td>↕</td>
        <td>向上或向下</td>
    </tr>
